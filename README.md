# 🚀 AI-Based Startup Valuation System

A lightweight web application that predicts startup valuation instantly using key financial inputs. Built using pure HTML, CSS, and JavaScript with a clean and responsive UI.


## 📌 Features

- 🔮 Instant startup valuation prediction
- 📊 Inputs: Revenue, EBITDA, Market Cap, Growth Rate
- 🎨 Smooth animated welcome screen
- 📱 Fully responsive design
- 💰 Output displayed in Indian Rupees (₹ Crores)
- ⚡ Fast and lightweight (no frameworks)


## 🛠️ Tech Stack

- HTML5
- CSS3 (Animations & Transitions)
- Vanilla JavaScript

## ⚙️ How It Works

The valuation is calculated using a simplified financial model based on:

- Revenue multiplier
- EBITDA margin impact
- Growth rate influence
- Market cap scaling

> Note: This is a basic predictive model for demonstration purposes and not for real financial decisions.
